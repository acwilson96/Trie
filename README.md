# Trie
Trie implementation in C++. Built using CMake and Google Test. 

# Installation
```
git clone https://github.com/acwilson96/Trie
cd Trie
mkdir build
cd build
cmake ..
make
make install
```

# Development/Testing
```
git clone https://github.com/acwilson96/Trie
cd Trie
mkdir build
cd build
cmake ..
make
make test
```
